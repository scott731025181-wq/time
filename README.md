# 静谧时光 - Minimal Focus

极简淡雅风格的待办 + 番茄计时 + 记录 + 习惯 Web 应用，按《静谧时光》PRD 设计，数据全部保存在浏览器本地，无需登录、无需后端。

## 功能概览

- **今日首页**：问候语、今日任务、习惯快速打卡、快捷入口「开始专注」「快速记录」
- **任务**：收件箱 / 今日 / 计划 / 已完成 分段，支持日期、分类、优先级，完成与删除
- **专注**：25 分钟番茄钟，开始 / 暂停 / 重置，今日专注统计
- **记录**：按时间线展示笔记，快速记录
- **习惯**：创建习惯、每日打卡、连续天数
- **我的**：已完成任务数、专注分钟、习惯连续天数；外观（浅色/深色/跟随系统）、强调色（静谧蓝/薄荷绿/薰衣紫/珊瑚粉/暖阳橙）

## 使用方式

1. 用浏览器打开 `index.html`（建议 Chrome / Edge / Safari）。
2. 所有数据保存在本机 `localStorage`，关闭页面后仍会保留。

### 在手机上使用

**方式一：同一 WiFi 下用电脑当服务器（适合在家/办公室）**

1. 电脑和手机连**同一个 WiFi**。
2. 在电脑上打开「静谧时光」文件夹，在地址栏输入 `cmd` 回车，打开命令行。
3. 输入并执行：
   ```bash
   python -m http.server 8080
   ```
   （若没有 Python，可安装 [Node.js](https://nodejs.org/) 后执行：`npx serve -p 8080`）
4. 查看电脑本机 IP：
   - Windows：在命令行输入 `ipconfig`，找到「IPv4 地址」，如 `192.168.1.100`
   - Mac：系统设置 → 网络 → 查看 IP
5. 手机浏览器打开：`http://电脑IP:8080`，例如 `http://192.168.1.100:8080`。
6. 在手机浏览器菜单里选择「**添加到主屏幕**」或「**安装应用**」，即可像 App 一样从桌面打开。

**方式二：部署到网上（手机随时可用，无需开电脑）**

1. 将整个「静谧时光」文件夹上传到任意**静态网站托管**服务，例如：
   - [Netlify](https://www.netlify.com/)：把文件夹拖进页面即可生成一个网址
   - [Vercel](https://vercel.com/)：同上
   - [GitHub Pages](https://pages.github.com/)：把项目放进仓库，在设置里开启 Pages
2. 用手机浏览器打开生成的网址。
3. 在浏览器菜单选择「**添加到主屏幕**」或「**安装应用**」，之后从桌面图标进入即可。

**说明**：数据存在**当前设备**的浏览器里，换手机或换浏览器需要重新添加任务/习惯；同一台手机添加主屏幕后，数据会保留。

## 文件说明

- `index.html`：页面结构
- `styles.css`：样式（PRD 色彩与组件规范）
- `app.js`：逻辑与本地存储
- `manifest.json`：PWA 配置，用于「添加到主屏幕」

## 技术说明

- 纯前端实现，无构建步骤，无 Node/npm 依赖。
- 数据键：`quietTime_tasks`、`quietTime_notes`、`quietTime_habits`、`quietTime_habitLogs`、`quietTime_focusRecords`、`quietTime_settings`。
